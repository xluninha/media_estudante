# Calculadora de Média

Um site simples e rápido para calcular a média de notas dos alunos e verificar se foram aprovados.

##  Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript

## ⚙️ Como Funciona
1. Introduza as notas do aluno nos campos indicados.
2. O sistema calcula a média.
3. O site mostra a nota final e diz se o aluno está **Aprovado** ou **Reprovado**.

## 📦 Como rodar na sua máquina
1. Baixe os arquivos do projeto para o seu computador.
2. Abra o arquivo `index.html` em qualquer navegador de internet
